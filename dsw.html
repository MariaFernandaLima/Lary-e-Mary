Convidando colaboradores para um repositório pessoal

Você pode convidar usuários para se tornarem colaboradores em seu repositório pessoal.

Repositórios de propriedade de uma organização podem conceder acesso mais granular. Para mais informações, consulte Permissões de acesso no GitHub.

Nota: O GitHub limita o número de pessoas que podem ser adicionadas a um repositório dentro de um período de 24 horas. Se você exceder esse limite, aguarde 24 horas ou crie uma organização.
